Formats des rapports Rmarkdown

Les rubriques “Formats Rdm” et “Rmd legers” permettent d’appliquer differentes mises en forme a un rapport basique (texte + tableau + graphique + image) ainsi qu’a la barre de navigation du site.

Les 3 types de rapports presentes ici sont :

  • les “html document” qui sont les rapports Rmarkdown usuels mais avec un theme a selectionner et une table des matieres flottante en option. Le theme que j’ai choisi pour le site est “spacelab”, mais j’aime bien aussi le theme “flatly”
  • les rapports qui utilisent les formats du package rmdformats, avec une mise en forme tres typee et la table des matieres flottante par defaut
  • les rapports bases sur les formats du package prettydoc, avec une belle mise en forme mais moins de fonctionnalites : la table des matieres n’est pas flottante et la barre de navigation du site n’est pas supportee

Les rapports prettydoc produits pesent autour de 100Ko, pour les rapports usuels on doit compter 1Mo et avec rmdformats on peut depasser 2Mo.

Les deux formats que j’utilise le plus :

  • le theme cayman du package prettydoc pour ma newsletter : les html produits sont legers et n’encombrent pas la boite mail de mes destinataires, et la taille de la police rend la lecture plus agreable
  • le format readthedown du package rmdformats pour des longs documents de type “bonnes pratiques” ou mes supports de formation : la table des matieres flottante est indispensable pour une navigation fluide et sa mise en forme tres structuree fait toujours forte impression
