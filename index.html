Mes newsletters Pyriodique / HebdomadR

J’anime depuis 2015 une newsletter reguliere a destination de mes collegues datascientists, une par semaine quand l’actualite est riche, moins frequemment dans les periodes plus creuses ou quand le temps me manque. C’est a partir de 2018 que je l’ai realisee au format Rmarkdown, une partie de ces documents sont archives ici. Lorsque le code n’est pas present dans la page (ex : flexdashboards), il peut etre recupere sur mon compte Github.
